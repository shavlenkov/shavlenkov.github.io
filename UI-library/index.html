Подключение в HTML-файл

Чтобы подключить UI-library нужно:

1. Загрузить UI-library.

2. Вставить в head этот код:

<--Grid-->
<link rel="stylesheet" href="UI-library/CSS/grid.css" type="text/css"/>

<--Button-->
<link rel="stylesheet" href="UI-library/CSS/button.css" type="text/css"/>

<--Modal-->
<link rel="stylesheet" href="UI-library/CSS/modal.css" type="text/css"/>

<--Burger-menu-->
<link rel="stylesheet" href="UI-library/CSS/burger-menu.css" type="text/css"/>

<--Carousel-->
<link rel="stylesheet" href="UI-library/CSS/carousel.css" type="text/css"/>

<--Table-->
<link rel="stylesheet" href="UI-library/CSS/table.css" type="text/css"/>

<--My-input-->
<link rel="stylesheet" href="UI-library/CSS/my-input.css" type="text/css"/>

3. Вставить в конец body этот код:

<--Modal-->
<script src="UI-library/JS/modal.js"></script>

<--Burger-menu-->
<script src="UI-library/JS/burger-menu.js"></script>

<--Carousel-->
<script src="UI-library/JS/carousel.js"></script>

<--Table-->
<script src="UI-library/JS/table.js"></script>

<--Vue.js-->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

<--My-input-->
<script src="UI-library/JS/my-input.js"></script>
